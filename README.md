# Colors

Colorset for everyone.

## 🔗 Links

- 📚 Docs: https://flightlesslabs.github.io/colors
- 💻 GitHub: https://github.com/flightlesslabs/colors
- 📦 npm: https://www.npmjs.com/package/@flightlesslabs/colors
- 📝 Changelog: https://github.com/flightlesslabs/colors/blob/main/CHANGELOG.md

## 🚀 Installation

```bash
pnpm add @flightlesslabs/colors
```

## 🎨 Import Styles

Import global styles in your root component. use the root layout (+layout.svelte) for SvelteKit.

```ts
import '@flightlesslabs/colors-base.scss';
import '@flightlesslabs/colors.scss';
```
